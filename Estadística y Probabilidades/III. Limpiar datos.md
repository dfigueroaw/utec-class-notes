Existen distintas técnicas de imputación para completar datos faltantes (NA) en un dataframe.
**Numéricas**
- Media -> Normalmente se toma esta cuando la media es representativa (Analizando el coeficiente de variación, normalmente se toma cuando es menor que 0.1, pero podemos tomar 0.3)
- Mediana -> Tomamos cuando el coeficiente de variación es mayor que 0.3 pero menor que 0.6
**No numéricas**
- Moda
